Homebridge Link Required

To use this Google Assistant action you must first install and configure the homebridge-gsh plugin in your local Homebridge instance.

You are seeing this page because your instance is not currently connected to the Homebridge GSH servers.

If you have already configured the homebridge-gsh plugin, please ensure you have restarted your Homebridge instance and signed in to this action using the same account you used to link your Homebridge instance.
